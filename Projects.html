Rotating ASCII Cube Animation

This project showcases an animated 3D ASCII representation of rotating cubes rendered directly in the terminal. Using simple 3D projection techniques and a z-buffer algorithm, the cubes appear to rotate in 3D space, creating a fascinating depth effect.

Key Features:

  • Multiple rotating cubes with different sizes and positions.
  • ASCII art surfaces represented by characters like @, $, ~, #, ;, +.
  • Real-time animation with smooth updates in the terminal.
  • Cross-platform compatibility for Unix-like systems and Windows.

How It Works: The program projects 3D cube vertices to 2D screen coordinates using trigonometric transformations. Rotation angles control the cubes' movement around the X, Y, and Z axes, and a z-buffer ensures proper depth representation.

Tech Stack: Written in C with ANSI escape codes for terminal rendering.

Check it out on GitHub: ASCII-cubespin

Custom Rules Cellular Automata

This project is an enhanced version of Conway's Game of Life, written in C, with customizable rules that allow users to define how cells evolve over time. It provides a flexible framework for experimenting with different cellular automaton rules, offering insight into how varying conditions influence the behavior of the grid and the emergence of patterns.

Key Features:

  • Classic Conway's Game of Life implementation with custom evolution rules.
  • Ability to modify birth and survival conditions.
  • Configurable grid size and customizable neighborhood definitions.
  • Command-line interface for easy rule modifications.
  • Efficient C implementation for fast simulations, even with large grids.

How It Works: Users can define their own set of rules for the simulation, modifying how cells "live" or "die" based on the number of neighbors and customizable conditions. The grid boundaries and neighborhood definitions are also fully adjustable, making it possible to simulate a wide variety of scenarios.

Tech Stack: Written in C with a modular structure for grid manipulation, rule management, and visualization.

Check it out on GitHub: Cellular Automata Project
